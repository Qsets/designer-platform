# 设计师与业主对接平台

这是一个基于React的设计师与业主对接平台前端应用。

## 功能特性

- 用户认证和授权
- 项目管理和展示
- 消息系统
- 个人资料管理
- 仪表盘数据展示

## 技术栈

- React 18
- TypeScript
- Ant Design
- CSS3

## 项目结构

```
frontend/
├── src/
│   ├── pages/          # 页面组件
│   ├── components/     # 通用组件
│   └── ...
```

## 开发说明

本项目使用现代化的前端技术栈，提供良好的用户体验和开发体验。
